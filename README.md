# 🛒 ShopEasy - E-Commerce Product Listing

A modern and responsive **E-Commerce Product Listing Website** built using **HTML, CSS, and JavaScript**. The website allows users to browse products, search items, filter by category, and sort products by price through a clean and interactive interface.

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/YourUsername/ecommerce-product-listing.git
```

### Open Project Folder

```bash
cd ecommerce-product-listing
```

### Run Local Server

Using Python

```bash
python -m http.server 8000
```

or

```bash
py -m http.server 8000
```

Open your browser and visit:

```
http://localhost:8000
```
